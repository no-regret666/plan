# 基于 `GNU/Linux` 操作系统，编程实现 `ls` 命令的 **部分** 功能．

## 目标

- 实现 ls 的 -a、-l、-R、-t、-r、-i、-s 参数，并允许这些参数任意组合．
- `-R` 需要通对 `/` 的遍历测试
- 界面美观（输出对齐、与颜色显示等）
- 无资源与内存泄露

## 命令输入示例

```c
ls -a -l
ls -a /home
ls -alR /
ls -t
ls -ai -t /home
ls -i /home /etc
```

注： 命令输出示例请参考 `ls` 命令
